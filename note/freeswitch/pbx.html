  <extension name="inbound">
    <condition field="destination_number" expression="^(\d{8,12})$">
      <action application="pre_answer"/>
      <action application="park"/>
    </condition>
  </extension>

解决设置多个号码外呼

  • 设置hangup_after_bridge=true不会导致不呼第二个
  • ignore_early_media=true：告诉bridge在收到早期媒体的时候不去连接被叫，只有在真正的answer才去连接。
  • originate_continue_on_timeout=true：书上的说要加，测试是不加也行。
    <extension name="example">
  <condition field="destination_number" expression="^(\d{8,12})$">
    <action application="pre_answer"/>
    <action application="set" data="hangup_after_bridge=true"/>
    <action application="set" data="ringback=/home/voices/rings/common/ring.wav"/>
    <action application="set" data="sip_h_Diversion=<sip:$1@ip>"/>
    <action application="bridge" data="{ignore_early_media=true,originate_continue_on_timeout=true}[leg_timeout=15]sofia/gateway/  zqzj/13675017141|[leg_timeout=25]sofia/gateway/zqzj/83127866"/>
    <!--<action application="park"/>-->
  </condition>
</extension>
  • 一下来自《freeswitch cookbook》对各种超时用法的表述
    • call_timeout: Sets the call timeout for all legs when using bridge
    • originate_timeout: Sets the call timeout for all legs when using originate
    • leg_timeout: Sets a different timeout value for each leg
    • originate_continue_on_timeout: Specifes whether or not the entire bridge or originate operation should fail if a single call leg times out
